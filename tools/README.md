# Tools 目录说明

本目录存放了用于维护、测试和生成 Lumex/Lumex/Stash 配置文件的各种辅助脚本和测试数据。

## 目录内容

- **Python 脚本 (*.py)**: 用于抓取规则、测试延迟、检查节点状态、提取配置等自动化任务。
- **HTML 文件 (*.html)**: 用于本地测试或作为抓取脚本的输入数据（如 DuckDuckGo 搜索结果、Stash Wiki 页面等）。
- **JSON 数据 (*.json)**: 辅助脚本运行所需的静态数据（如 eleases.json）。

## AI 助手使用指南

当用户要求进行以下操作时，请优先在此目录中寻找或创建相关脚本：

1.  **规则更新与抓取**: 如果需要从网页或 API 提取新的分流规则。
2.  **配置测试**: 如果需要编写脚本来验证配置文件的正确性或测试节点的连通性。
3.  **批量处理**: 如果需要对大量的规则或节点进行格式转换、去重等操作。

**注意**: 核心的配置文件（如 Lumex.yaml, Stash.yaml 等）应保持在项目根目录，不要移动到此文件夹中。
